# optimization_armijo-_gabrielmanhaes
This repository contains MATLAB scripts and algorithms developed for the study and implementation of the Armijo condition in numerical optimization. This work is part of an ongoing undergraduate scientific research project at the Federal University of Technology - Paraná (UTFPR).
